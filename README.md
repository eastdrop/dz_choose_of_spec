## Для решения поставленной задачи создаем метод **string[] ModifyArray(string[] array)**, принимающий строковый массив и который выполняет следующие действия:
1. Проходит по всем элементам массива, и считает количество элементов, удовлетворяющих условию *( <= 3)*
2. Перезаписываем элементы не подходящие под условие в виде -> *""* 
3. Заводит новый массив длиной равной количеству удовлетворяющих условию элементов
4. Записывает в новый массив элементы с количеством символов меньше или равным трем.
5. Возвращает новый массив элементов
 Следующий метод **void PrintArray(string[] arr)** позволяет нам вывести в консоль массив в виде [ , ...., ].
